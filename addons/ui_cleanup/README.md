# UI Cleanup

## Overview

This module provides a cleaner, more professional Odoo interface by removing unnecessary branding and menu items.

## Features

### 1. Clean Login Screen

- Removes "Powered by Odoo" branding from the login page
- Keeps only essential elements (database manager link if needed)

### 2. Streamlined User Menu

Removes the following items from the user menu (top right corner):

- Documentation
- Support
- Shortcuts
- Odoo.com account
- Install PWA (Progressive Web App)
- Tour onboarding
- Menu separators

## Installation

1. Copy this module to your Odoo addons directory
2. Update the apps list in Odoo
3. Install "UI Cleanup" from the Apps menu

## Configuration

No configuration needed. The module works automatically once installed.
